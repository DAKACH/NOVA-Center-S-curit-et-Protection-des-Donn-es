# NOVA Center – Security and Data Protection

##  Project Description
**NOVA Center – Security and Data Protection** is a web-based project designed to present services and information related to cybersecurity and data protection.  
The project combines a simple front-end interface with a Python-based back-end.

---

##  Project Structure

###  Front-End
- `index.html` – Home page  
- `apropos.html` – About us page  
- `formations.html` – Training programs  
- `contact.html` – Contact page  
- `style.css` – Website styling  
- `main.js` – Client-side interactions  

###  Back-End
- `app.py` – Main backend application (Python)  
- `backup.py` – Backup script  
- `database.sql` – Database structure  

---

##  Requirements
- Python 3.x  
- Modern web browser (Chrome, Firefox, Edge)

For more details, see `REQUIREMENTS.md`.
